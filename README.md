# 阿呆禅 - 商城小程序

#访问小程序码


   ![image](https://github.com/wudan1874/mall/blob/master/readme-img/QRcode.jpg)



呆·优选是基于微信平台开发的一款电商类微信小程序，依托微信强社交优势，一键转发和分享功能，轻松玩转社交电商。其主要功能有 商品预览、商品分类、商品购买、加入购物车、微信支付、地址管理、查看订单等功能。
